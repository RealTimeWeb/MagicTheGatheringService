A Magic the Gathering Card

  • id — A unique id that identifies this card.

  • name — The name of this card.

  • mana-cost — The Mana cost of this card.

  • converted-mana-cost — The converted mana cost.

  • types — Card’s types, usually at least one of "artifact", "creature", "enchantment", "instant", "land", "planeswalker", "sorcery", or "tribal". Cards can also have a supertype and/or subtype.

  • texts — Any text blocks on the card.

  • flavors — Any flavor texts on this card.

  • power — The power (http://mtg.wikia.com/wiki/Power) of this card

  • toughness — The toughness (http://mtg.wikia.com/wiki/Toughness) of this card

  • watermark — The watermark of this card

  • set — The expansion set that this card belongs to.

  • rarity — How rare this card is, typically either "uncommon", "common", or "rare".

  • all-sets — All the expansion sets that this belongs to.

  • number — The Card Number.

  • artist — The name of the artist for the card’s artwork.

  • rating — The card’s voted upon rating.

  • votes — The number of times this card has been voted on.
