Creates a new Card

Parameters:
  • self (Card) – This object
  • id (string) – A unique id that identifies this card.
  • name (string) – The name of this card.
  • mana_cost (listof string) – The Mana cost of this card.
  • converted_mana_cost (string) – The converted mana cost.
  • types (listof string) – Card’s types, usually at least one of “artifact”, “creature”, “enchantment”, “instant”, “land”, “planeswalker”, “sorcery”, or “tribal”. Cards can also have a supertype and/or subtype.
  • texts (listof string) – Any text blocks on the card.
  • flavors (listof List(String)) – Any flavor texts on this card.
  • power (string) – The power (http://mtg.wikia.com/wiki/Power) of this card
  • toughness (string) – The toughness (http://mtg.wikia.com/wiki/Toughness) of this card
  • watermark (string) – The watermark of this card
  • set (string) – The expansion set that this card belongs to.
  • rarity (string) – How rare this card is, typically either “uncommon”, “common”, or “rare”.
  • all_sets (listof ExpansionPrint) – All the expansion sets that this belongs to.
  • number (string) – The Card Number.
  • artist (string) – The name of the artist for the card’s artwork.
  • rating (string) – The card’s voted upon rating.
  • votes (string) – The number of times this card has been voted on.
